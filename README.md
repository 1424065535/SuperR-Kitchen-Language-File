  1.复制chinese_srk.py到/tools/language
  
  2.或者在第一次u运行 ./superr 后  选择 d (Download language) ，选择4 下载Chinese 即可
	
  3.如果已经运行了，运行后-6--2--回车--r--d(下载语言文件)--选择chinese(本操作是重设语言)即可完成汉化
